# Teacher Manual  
## Delivering the NerdCommand Beginner Options Desk with OptionScope

**For:** Michael Chapman (instructor) + future NerdCommand coaches  
**Course length:** 4 weeks · or 11 video episodes · or 1-day workshop (compressed)  
**Software:** OptionScope local or hosted demo  

---

## 1. Instructor stance

| Do | Don’t |
|----|-------|
| Teach process and defined risk | Promise monthly income |
| Show OptionScope live | Hide DEMO vs LIVE |
| Celebrate size 0 honesty | Override risk ceilings “for demo drama” |
| Use paper first | Live YOLO for engagement |
| Repeat disclaimers | Soften laws when students pressure |

**Opening line every live class:**  
“This is education. Options can lose money. OptionScope does not place trades. You own every click.”

---

## 2. Prerequisites for students

- Computer + browser  
- OptionScope access (install via `install.txt`)  
- Optional: broker paper account  
- Notebook or OptionScope Journal  
- Calculator OK; brain does sizing

**Not required:** prior options license knowledge (you will teach vocabulary).

---

## 3. Classroom setup (software)

1. `npm run dev` → http://localhost:3000  
2. Projector / share: **dark theme**  
3. Settings: seed **$500** (or student real paper equity)  
4. MARKET_DATA: demo OK for class 1–2; live Alpaca for class 3+ if keys  
5. Open tabs ready: Education, Command, Scanner, Lab, Journal  
6. Never display secret keys

---

## 4. Lesson plans (week-by-week)

### Week 1 — Laws & desk map  
**Outcomes:** Students recite 5 laws; navigate all tabs.  
**Time:** 90 min  

| Min | Activity | Tool |
|-----|----------|------|
| 0–10 | Open + disclaimer | Spoken |
| 10–30 | Book Ch.1–3 discussion | 01 book |
| 30–50 | Live tour of nav | OptionScope |
| 50–70 | Lab A chain literacy | Trade Lab |
| 70–85 | Journal first line | Journal |
| 85–90 | Homework assign | — |

**Homework:** Read strategies 1–2 overview; do not live trade.

### Week 2 — Four safer strategies  
**Outcomes:** Sketch payoffs; state max loss formula.  
**Time:** 120 min  

| Min | Activity |
|-----|----------|
| 0–15 | Recap + quiz (5 laws) |
| 15–45 | Strategy 1–2 deep dive + Scanner |
| 45–75 | Strategy 3–4 + Compare |
| 75–110 | Paper build each structure once |
| 110–120 | Debrief max loss aloud |

**Homework:** 5 paper checklists; no live.

### Week 3 — Full ritual & green lights  
**Outcomes:** Unassisted daily ritual; identify red lights.  
**Time:** 90 min  

- Run Ch.7 ritual timed (15 min)  
- Ch.8 green/red table drills  
- 07 profitable-shape checklist on 3 tickers  

### Week 4 — Review, expectancy, graduation  
**Outcomes:** Present journal stats; optional micro-live policy.  
**Time:** 90 min  

- Share anonymized paper results  
- Evolve optional demo (hive explanation)  
- Graduation criteria check  
- Marketing-safe CTA only if cohort paid product

## 5. Live demo scripts (copy/paste)

### Demo: Bull put credit (10 min)
1. Settings equity visible.  
2. Scanner → bull put credit → pick liquid name.  
3. Lab load chain.  
4. Apply structure / brain.  
5. Say max loss = (width − credit) × 100 × contracts.  
6. Show checklist.  
7. **Do not** place broker order in class unless paper.  
8. Journal plan.

### Demo: Size 0 coach (5 min)
1. Equity $500.  
2. Force wide/expensive structure if needed.  
3. Read coach text out loud.  
4. “This is the product working.”

## 6. Assessment

| Type | Pass bar |
|------|----------|
| Quiz (laws + vocab) | ≥80% |
| Practical (checklist build) | Complete + correct max loss |
| Journal sample | ≥10 paper plans |
| Conduct | No revenge sizing stories as “strategy” |

**Fail → remediate:** re-run Week 1–2; no live endorsement.

---

## 7. Handling tough questions

| Question | Answer |
|----------|--------|
| “How much can I make?” | “Unknown. We optimize process and risk. Sample size first.” |
| “Is AI picking trades?” | “Rules + models rank structures. You decide and execute.” |
| “Why not CSP?” | “Cash lock vs seed equity. Empire blocks CSP as primary at seed.” |
| “Can I 0DTE?” | “Not in beginner core. Defined multi-DTE first.” |
| “Will you manage my money?” | “No. Education and software only.” |
