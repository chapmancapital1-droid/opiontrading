# Shape of a Profitable Option Trade  
## What to look for on OptionScope before you risk a dollar

**Profitable shape ≠ guaranteed profit.**  
It means the trade is **built like an operator trade**: defined, sized, liquid, journaled, executable.

Print this page. Use it every session.

---

## 1. One-screen definition

A NerdCommand “A+ shape” trade has:

1. **Thesis** in one sentence  
2. **Structure** from the four safer strategies (or paper advanced)  
3. **Defined max loss** in dollars  
4. **Size** allowed by empire/seed rules  
5. **Live chain** marks (if live money)  
6. **Checklist** complete  
7. **Exit plan** written  
8. **Emotion** calm  

If any critical item fails → **no trade**.

## 3. Number checks (speak aloud)

For **credit spreads / condor wings**:
```
Max loss $ = (width − credit) × 100 × contracts
Max profit $ = credit × 100 × contracts
Is max loss ≤ my per-trade ceiling?
```

For **debit spreads**:
```
Max loss $ = debit × 100 × contracts
Max profit $ = (width − debit) × 100 × contracts
Is debit tuition worth the thesis?
```

Ceiling example (seed discipline):  
If equity $500 and hard risk ~0.5–1%, many structures **cannot** size — correct outcome is **flat**.

## 4. Example “A+ shape” narrative (illustrative only)

> “SPY-ish liquid ETF, no earnings this week, Bias not violently opposed,  
> bull put credit 30-ish DTE, short ~0.25–0.30 delta guide,  
> width fits, credit decent vs width, max loss $X under ceiling,  
> LIVE chain, checklist copied, journal plan written, calm.”

**Not A+:**  
“Cheap OTM calls, DEMO marks, size overridden, earnings tomorrow, revenge after loss.”
