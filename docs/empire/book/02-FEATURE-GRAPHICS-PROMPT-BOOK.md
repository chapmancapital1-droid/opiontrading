---

## How to use this book

1. Copy a **Prompt** block into your image model.  
2. Always include the **Negative prompt**.  
3. Prefer **dark cockpit** variants for analysis surfaces; light only when documenting Settings/Education reading mode.  
4. Use **Aspect ratio** as suggested; crop later for slides vs book full-bleed.  
5. Place the **Caption** under the figure in the personal book / field manual.  
6. If the model invents a green **BUY / Place Trade** button, regenerate — brand forbids it.  
7. Badges in frames should read: **LIVE · DEMO · PAPER · MODEL EST. · HALT** as appropriate — never unlabeled “live looking” demo.
